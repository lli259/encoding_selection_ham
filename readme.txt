This folder contains
1.csv folder
Insider the folder, there are two files
1.1 algorithm_runs.csv: encoding performance data.
1.2 feature_values.csv: selected feature values from all features.

2.main.py: 
A python3 code for encoding selection.

how to run: 
put csv folder above in the same folder, and run
'python main.py'
results will be printed in the terminal.
It will take a while to tune the hyperparameters at first run.

The package required are pandas,numpy,matplotlib,sklearn
